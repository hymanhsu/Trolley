Trolley
====

Http+Json App Server Development Framework
<br>
整合了一些开源库(有些在deps里面，有些直接集成的源代码)，提供基于Http+Json的服务接口的快速开发的能力。


<br><br>


#功能介绍


> * 基于Http+Json的Server的开发搭建
> * 带有并发能力的Http Client
